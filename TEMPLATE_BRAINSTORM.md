## Template Characteristics to Consider

### Style Variations
- **Brutalist**: Raw, bold, unconventional
- **Glassmorphism**: Frosted glass effects
- **Neomorphism**: Soft UI with shadows
- **Minimalist**: Clean, spacious, simple
- **Maximalist**: Rich, bold, layered
- **Retro/Vintage**: Nostalgic design elements
- **Futuristic**: Sci-fi, tech-forward

### Layout Patterns
- **Full-width hero**: Large hero sections
- **Split screen**: Divided layouts
- **Grid-based**: Card/tile layouts
- **Asymmetric**: Non-standard layouts
- **Scroll-based**: Story-driven scroll
- **One-page**: Single page scroll sites

### Interaction Patterns
- **Cursor effects**: Custom cursors, followers
- **Parallax**: Depth effects on scroll
- **Horizontal scroll**: Side-scrolling sections
- **Hover reveals**: Content on hover
- **Magnetic elements**: Elements that follow cursor
- **3D transforms**: CSS 3D effects
